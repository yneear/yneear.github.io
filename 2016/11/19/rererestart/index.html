叕开始写了。

又开始写博客了。

在两年前就开始使用 WordPress 搭建第一个自己的博客（ynearu.me），当时还为这个博客写一个主题，主题使用世界知名框架 Bootstrap 来写的。第一个博客大概只写五六篇，关于技术的文章很少，大多写的是关于学校里的一些事情和人。

之后渐渐开始学习和使用 Node.js，当时搜一下 Node.js 有哪些博客平台，了解到 Ghost 这一博客平台，然后又开始搭建新的博客（yneear.me），这次并没有写主题。文章也只写了几篇关于平时遇到的一些技术问题解决方案的文章，其他多数是从上一个博客那里搬运过来。

不能坚持写博客还是因为懒，没有能坚持。写博客对知识的巩固和语言组织能力都有很大帮助，也是所谓的“输出”（虽然懂得很多道理，但还是懒）。以前写完文章总有一种和写完代码一样的成就感，希望之后能坚持并继续感受这种成就感。

最近已经是“大五”上后半期了，想想自己运气还算不错的，认识了一些很好的朋友，让我可以在“大五”期间继续开心地写代码。在获得一些资源后，便从七月开始在 WhatYouNeed 从零开始搭建各个网站应用，虽然对关于整个网站架构方面的问题比较生疏，也没有什么很成熟的方案，多请教一些人来获得解决方案，但还是继续边学边做吧。

前些天突然觉得自己做的一些项目，做完之后，过程中的一些问题还是会经常在脑海里浮现，便又有了写博客的冲动。这一次选择了 Hexo，原因还是因为太懒（？？？），不想再去弄域名这些东西，没怎么考虑就选择 GitHub Pages 结合 Hexo 搭建博客的方案。以前只看也没用过，用起来确实很方便，也不用去再搞域名（yneear.github.io）。

前几天，想在宿舍继续写代码，结果发现 Windows 10 和之前的 Windows 系统一样，Node.js 还是那么难配，terminal 也很难用，萌生了装 Ubuntu 双系统的想法。马上就开始下载 Ubuntu 的镜像文件，装双系统有些开机引导方面的问题，搞了两天到凌晨两三点才睡。但现在用起来真的很舒爽，和 Mac 的感觉还是有一些区别的，特别是 Markdown 编辑器 Haroopad，在 Mac 上似乎没发现过有类似的编辑器，让我这么想打字。系统的体验也很棒，五笔输入法的感觉好像也比 Mac 上好用很多。以后下班回来可以继续写一些东西。

想写的东西还是想以技术为主，平时一些事情如果有特别能启发自己的东西也会记录一下，话都少说了再不写出来迟早要完，总之就这样重新开始吧。
